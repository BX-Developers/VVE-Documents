	数据模板是一段以NBT形式储存的，关于一种载具全部属性数据的复合标签。设计载具即为设计载具的数据模板的过程。

示例载具模板库：
	使用<接口函数> vehicles:_class即可导入示例载具模板库。其中包含了storage math:class vehicles这个列表。列表的
每一项即为一种示例载具的数据模板。

解析数据模板：
	使用<接口函数> vehicles:_spawn即可解析一个数据模板，将其表示的载具实例化。

示例：
	#导入示例载具模板库(只需执行一次)
	function vehicles:_class

	#生成载具
	data modify storage math:io input set from storage math:class vehicles[{id:"first_car"}]
	data modify storage math:io input.pos set value [85.5d,93.0d,45.5d]
	data modify storage math:io input.rotation set value [-90.0f,0.0f]
	function vehicles:_spawn

这个例子可以在坐标85 93 45的位置生成一辆朝向正东的示例车。

数据模板格式：
----------------------------------------------------
		键名:id
		类型:string
		说明:用于索引该型号载具
		参考值:略
		必要性:无
----------------------------------------------------
		键名:ctrl_type
		类型:int
		说明:载具的<操作模式>
		参考值:2
		必要性:需要玩家操控必填
----------------------------------------------------
		键名:mass
		类型:double
		说明:载具的惯性质量,影响运动量改变快慢
		参考值:5.0d
		必要性:必填
----------------------------------------------------
		键名:back
		类型:double
		说明:倒车速度
		参考值:1.25d
		必要性:0,1,2,3,4,6号操作模式必填
----------------------------------------------------
		键名:v_max
		类型:double
		说明:最大速度
		参考值:8.0d
		必要性:必填
----------------------------------------------------
		键名:r_max
		类型:double
		说明:最大偏航角速度
		参考值:45.0d
		必要性:0,1,2,3,4,5,6号操作模式必填
----------------------------------------------------
		键名:u_max
		类型:double
		说明:影响俯仰速度
		参考值:30.0d
		必要性:3,5号操作模式必填
----------------------------------------------------
		键名:fly_v
		类型:double
		说明:起飞所需速度/螺旋桨角速度
		参考值:13.5d
		必要性:1,4,5号操作模式必填
----------------------------------------------------
		键名:fly_u
		类型:double
		说明:起飞时设定的目标仰角(飞机会逐渐接近这个仰角)
		参考值:35.0d
		必要性:1,4号操作模式必填
----------------------------------------------------
		键名:bounce
		类型:double
		说明:碰撞弹性，碰撞后前速度大小比值
		参考值:0.3d
		必要性:有碰撞点必填
----------------------------------------------------
		键名:low_sp
		类型:double
		说明:低速判定的临界速度,影响在斜面上飞出去还是贴着斜面走。影响<状态标签> vehicle_LS判定。
		参考值:6.5d
		必要性:无,若不填则为参考值6.5d
----------------------------------------------------
		键名:rot_um
		类型:double
		说明:载具在空中时的最大俯仰角
		参考值:30.0d
		必要性:0,2号操作模式必填
----------------------------------------------------
		键名:rot_rm
		类型:double
		说明:载具在空中时的最大横滚角
		参考值:30.0d
		必要性:0,1,2,4,5号操作模式必填
----------------------------------------------------
		键名:braking
		类型:double
		说明:刹车加速度
		参考值1.25d
		必要性:0,1,2,3,4,6号操作模式必填
----------------------------------------------------
		键名:gravity
		类型:double
		说明:重力加速度
		参考值:3.0d
		必要性:非悬浮必填
----------------------------------------------------
		键名:friction
		类型:double
		说明:摩擦系数，载具在地面/水面上时速度大小每gt乘以的系数。
		参考值:0.959d
		必要性:非悬浮必填
----------------------------------------------------
		键名:gravity_point
		类型:list_double
		说明:重心位置,辅助gravity_vector,局部坐标形式。
		参考值:[0.0d,0.0d,0.0d]
		必要性:无,若不填则全为0
----------------------------------------------------
		键名:gravity_vector
		类型:list_double
		说明:着陆点向量,局部坐标形式。gravity_vector[0]和gravity_vector[2]必须为正值。
		     将gravity_vector[0]和gravity_vector[2]分别取反,同时取反,可以得到3个镜像向量。
		     这四个向量分别加上gravity_point即为载具四个着陆点位置。
		参考值:[0.7d,-0.5d,0.9d]
		必要性:非悬浮必填
----------------------------------------------------
		键名:buoyancy_point
		类型:list_double
		说明:浮力点位置,用于决定船类载具浮出水面的部分
		参考值:[0.0d,0.2d,0.0d]
		必要性:6号操作模式必填
----------------------------------------------------
		键名:collision_points
		类型:list_(list_double)
		说明:碰撞点列表
		参考值:[[0.5d,0.0d,1.0d],[-0.5d,0.0d,1.0d]]
		必要性:无
----------------------------------------------------
		键名:cubes
		类型:list_cmp
		说明:<cube格式>的列表
		参考值:[{}]
		必要性:无
----------------------------------------------------
		键名:specs
		类型:list_cmp
		说明<spec格式>的列表,最多支持9项
		参考值:[{}]
		必要性:无
----------------------------------------------------
		键名:pos
		类型:list_double
		说明:载具生成位置,mc世界坐标形式
		参考值:[0.0d,0.0d,0.0d]
		必要性:无,若不指定,为_spawn函数执行位置
----------------------------------------------------
		键名:rotation
		类型:list_float
		说明:载具生成朝向,mc视角坐标形式
		参考值:[0.0f,0.0f]
		必要性:无,若不指定,为[0.0f,0.0f]
----------------------------------------------------
		键名:name
		类型:json转义形式string
		说明:载具名字
		参考值:'{"text":"test"}'
		必要性:无,若不指定,则为"vehicle"+载具序号
----------------------------------------------------
		键名:get_in_text
		类型:json转义形式string
		说明:引导文本的文本
		参考值:'{"text":"右键以进入载具"}'
		必要性:无,若不指定,则为"right click to get in"
----------------------------------------------------
